    Technologies and Sub-Programs workings :-

    Lining Icons:-

  •     First of all icons directory list is loaded into memory
  •     loop starts from 0 to listcount
  •     Data is retrieved from bmp file   
  •     New icon is made
  •     Icon's properties are set to data
  •     Icons are divided into two categories. ie Pre-Coordinated and others
  •     Pre-coordinated icons are placed according to data
  •     loop is ended
  •     Others non-coordinated icons are placed listwise

    USEI (Universal String Encryption by Image):-

  •     Firstly you need to open any icon in \Links\ directory
  •     Stretch it vertically
  •     You will see diffrent lines of diffrent colors
  •     Color determines the ascii value of string
  •     First of all the string value is split into alphabets
  •     A random number is generated
  •     A new blank picture is loaded
  •     The color of first pixel (0,0) is the random number
  •     The ascii values of alphabets is multiplied by random number
  •     Then the number is rendered in the picture horizontaly in form of colored pixels
  •     To retrieve the string the integeral value of pixel is divided by first pixel
  •     Please open crypt.bas for detail

    How To.. :-

    Make a new Icon

  •     Right click on dextop and select the choice, Enter data on the dialog
  •     Click on the icon to open the icon selector

    Change the properties of current icon

  •     Right click on dextop and select properties page
  •     You will see the list. Select any icon and change properties

    Lock an icon

  •     Right click on any icon and use your brain

    Skin Dextop

  •     Goto Customize and click Skin
  •     Now extend skin dialog by clicking buttons on left
  •     Select any skin by skinlist or exploring directories
  •     Remember that skin is a directory not a file

    Make a skin

  •     You will need pro softwares like Jasc paintshop or Adobe photoshop
  •     Explore the skins Directory
  •     Duplicate any skin folder
  •     Edit images in the new folder
  •     Apply skin by method above

    Start-up program

  •     Open the application directory and find config.ini
  •     Open the file in notepad
  •     Now find 'Run' in '[Main]'
  •     Give the 'Run' value the desired path
  •     It can run file of any extension

    Hide any running window

  •     Right click on any any taskbar item and goto Set Order -> Hide
  •     It will goto Hidden programs list
  •     To redo it right click on taskbar and goto Hidden Programs -> Window name
  •     You can also select multiple show

    Multiperform functions on a window

  •     Right click on any taskbar and select Multiple Perform
  •     Select widows by holding ctrl key
  •     Select any function

    Search Web

  •     Write any string on the 'Search Internet' textbox
  •     Hit Enter key, Select Google or planet-source-code.com

    Glass Any window

  •     Drag pointer from digital clock to any window and drop it
  •     A new scale will show, Change the scale to your value

    Change Username or password

  •     Default username is 'user' and password is 'pass'
  •     To change goto customize -> About -> User accounts
  •     Enter current password to proceed
